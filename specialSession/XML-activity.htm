  1. Give two advantages of XML over HTML.
    • One advantage of using XML is because you are using our own markup language, you are free to use whatever tags you prefer. The user is not restricted to specific tags to transfer data. Another advantage is how much XML has been embraced by the software community. This is because XML is easy to read and write, allowing it to be more readily used.

  2. What is the difference between well-formedness and validity?
    • An XML document is well-formed if it has the correct syntax. An XML document is valid if it conforms to a specified document type, such as DTD's and XSD's.

  6. What is an XML Schema and why use it?
    • An XML Schema describes the structure of an XML document, similar to the way a DTD can describe an XML document. The advantage of using an XSD over a DTD is because XSD's are more powerful. This is because XSD's are written in XML so they are easier to use; they support data types; and in addition, they support namespaces.

  8. Is it possible to have an XML document that is well-formed but not valid?
    • Yes it is possible to have a well-formed XML document that is not valid. The only requirements for it to be well-formed is that is it syntatically correct, such as in Address.xml. This, however, is not valid, because it does not have a second address line or country line, the zip code is too long, and the state is not one of the listed acceptable values.

  9. Is it possible to have an XML document that is valid, but not well-formed?
    • No it is not possible for an XML document to be valid, but not well-formed. This is because on of the requirements to be considered valid is that is not only has to adhere to the rules of an XSD or DTD, but it also must be syntatically correct.

  10. Write about XBRL or RSS
    • RSS stands for Rich Site Summary. It is used to transmit data of frequently updated information, such as blogs, news headlines, audio clips, and videos. XML is used to transfer the data because it allows it to be transmitted in a similar format so that machines can easily read and display the data provided.
